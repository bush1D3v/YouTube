# Aula 05 — Diretivas no Vue 3 (Projeto de Demonstração)

Projeto com **uma rota por diretiva** (cada diretiva em um arquivo `.vue` separado), pronto pra você rodar:

```bash
npm i
npm run dev
```

## Rotas
- `/` → Home
- `/v-if`
- `/v-show`
- `/v-for`
- `/v-bind`
- `/v-model`
- `/v-on`

## Estrutura
- `src/router/index.js` → rotas
- `src/views/*.vue` → um arquivo por diretiva
- `src/components/NavBar.vue` → navegação
